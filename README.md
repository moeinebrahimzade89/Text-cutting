# ✂️ Text Cutter

A simple Python script that takes a line of text from the user and prints only the part up to the **first period (`.`)**, including that period.

## 🛠 Technologies Used
- Python 3.x

## ✨ Features
- Accepts a text input from the user
- Finds the first occurrence of the period character (`.`)
- Extracts and prints the text from the beginning up to that period
- Very lightweight and easy to use

## 🚀 How to Run
1. Save the file as `Text cutting.py`
2. Open your terminal or command prompt
3. Run the program with:
```bash
python "Text cutting.py"
